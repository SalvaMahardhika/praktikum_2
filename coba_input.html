- Elemen type="radio" digunakan untuk membuat pilihan eksklusif, di mana hanya satu opsi yang dapat dipilih pada satu waktu.

Uji coba 3



Setelah uji coba dengan memberikan atribut "name" yang sama dan berbeda pada tag input checkbox, dapat disimpulkan bahwa:

- Jika atribut "name" memiliki nilai yang sama, maka hanya satu pilihan dapat dipilih pada satu waktu.

- Jika atribut "name" memiliki nilai yang berbeda, maka setiap pilihan dapat dipilih secara independen.

Uji Coba 4




- Elemen type="color" digunakan untuk memungkinkan pengguna memilih warna melalui antarmuka yang disediakan oleh browser.

- Elemen type="date" digunakan untuk memungkinkan pengguna memilih tanggal melalui kalender yang disediakan oleh browser.

- Elemen type="button" dengan atribut onclick dapat digunakan untuk menjalankan fungsi atau skrip JavaScript saat tombol tersebut diklik.

- Pada contoh di atas, saat tombol "Click Me!" diklik, akan muncul sebuah alert dengan pesan "Hello World!".

Uji Coba 5




Elemen HTML type="datetime-local" dan type="month" memberikan fleksibilitas dalam pengumpulan informasi waktu dan tanggal, sementara type="email" digunakan untuk memvalidasi alamat email pengguna.

Uji coba 6




- Elemen type="image" digunakan untuk menyertakan gambar sebagai tombol submit dalam formulir. Ketika gambar tersebut diklik, formulir akan dikirimkan seperti tombol submit biasa.

- Elemen type="file" memungkinkan pengguna memilih dan mengunggah file melalui antarmuka browser. File yang dipilih akan dikirimkan sebagai bagian dari formulir.

- Elemen type="hidden" digunakan untuk menyimpan nilai tertentu dalam formulir yang tidak ditampilkan kepada pengguna. Pada contoh ini, nilai "3487" disimpan dan akan dikirimkan saat formulir dikirimkan.

- Kegunaan dari masing-masing jenis input ini melibatkan pengiriman informasi ke server: type="image" untuk tombol submit berupa gambar, type="file" untuk mengunggah file, dan type="hidden" untuk menyimpan nilai tanpa menampilkannya pada formulir.

Uji coba 7



- Elemen type="number" membatasi input angka dalam rentang -50 hingga 10, tetapi pada praktiknya, nilai input hanya akan menerima angka antara 1 dan 5 dikarenakan batasan min dan max dalam contoh ini.

- Elemen type="range" memberikan kontrol slider yang memungkinkan pengguna memilih nilai dalam rentang 0 hingga 50.

Uji coba 8




- Elemen type="url" digunakan untuk memvalidasi dan memastikan bahwa input yang dimasukkan oleh pengguna adalah URL yang valid.

- Elemen type="search" digunakan untuk memasukkan kata kunci pencarian, dengan antarmuka yang dapat disesuaikan oleh browser.

- Elemen type="tel" digunakan untuk memvalidasi dan membatasi input untuk format nomor telepon tertentu, dalam contoh ini, menggunakan pola (pattern) "[0-9]{3}-[0-9]{2}-[0-9]{3}".

- Hasil uji coba menunjukkan bahwa elemen type="url" dan type="tel" berfungsi sesuai dengan tujuannya dalam memvalidasi dan membatasi format input, sementara elemen type="search" memberikan antarmuka pencarian yang dapat digunakan.

Uji coba 9



- Elemen type="time" memungkinkan pengguna memilih waktu menggunakan antarmuka yang disediakan oleh browser. Format waktu yang dapat dipilih bergantung pada implementasi browser.

- Elemen type="week" memungkinkan pengguna memilih minggu menggunakan antarmuka yang disediakan oleh browser. Pengguna dapat memilih minggu dengan menavigasi melalui kalender.
